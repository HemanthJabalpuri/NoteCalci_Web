About NoteCalci Web 🧮

NoteCalci Web is an offline-first browser document calculator. It combines the formatting elegance of markdown documentation with the calculations precision of isolated-scope compilers.

Open Source Inspirations & Heritage

🧮 Core Math Compiler (NerdCalci)

NoteCalci Web's compiler brain is ported from and modeled directly after NerdCalci by Vishal Telangre. We port its custom Lexer, recursive AST Parser, and write-isolated cascading variable functions closures line-by-line to ES6 JS parameters, guaranteeing 100% hermetic safety.

📝 Layouts & Workspace (Notepad Calculator)

Several visual features and calculations conveniences are adapted from Notepad Calculator by Steve Ridout. We integrated his absolute horizontal overlays, synchronized scrolling locks, notes sidebars, and renamer bars—as well as the elegant system of multi-line operator carry-on (curry), the ans preceding result keyword, the above_block contiguous variables list array, and the seamless mix of rich Markdown formatting prose inside the calculations sheet. We extend our deep gratitude to Steve.

Workbook Syntax Cheatsheet & Guides

Variables & Custom Functions

Predefine constants or declare isolated dynamic variables Cascading scopes:

  • Variables: price = 1500
  • Multi-params Functions: add(a, b) = a + b
  • Side-effects safety: modifications inside functions are strictly local.

Percentage Operations Shorthand

Compact formulas to compute discounts or increases dynamically without syntax clutter:

  • Slice of: 12% of 25000
  • Percentages off: 15% off 3000 (deducts 15%)
  • Inline addition: 1500 + 10%
  • Inline subtraction: 1500 - 20%

Multi-line Operator Carry-On

Start a line with standard binary operators to smoothly carry on active equations from the line above:

  • + 50 (adds 50 to previous answer)
  • * 2.5 (multiplies previous answer by 2.5)
  • Use the keyword ans referencing trailing answer anywhere.

Composite Lists & Array Math

Exposes preceding arithmetic blocks as arrays to process composite aggregates:

  • Retrieve rows array: expenses = above_block
  • List sums: sum(expenses)
  • List averages: avg(expenses)
  • List sizing metrics: length(expenses) / size(expenses)
