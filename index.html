Syntax Cheatsheet

Variables & Functions

  • x = 250 (Define variable)
  • total = x * 2.5 (Multiply variable)
  • sqrt(100) (Standard Root)
  • pow(2, 10) (Binary Power exponent)

Percentages shorthand

  • 20% of 15000 (Part computation)
  • 15% off 1000 (Discounted calculation)
  • 5000 + 10% (Dynamic increase inline)
  • 5000 - 20% (Dynamic decrease inline)

Incremental Compounds

  • x++ / x-- (Scalar steps)
  • x += 10 (Compound additions)
  • x *= 5 (Compound multiplications)

Block Aggregations

  • sum / total (Current block sum)
  • avg / average (Current block avg)
  • grand_total (Document totals)
  • prev / last / _ (Preceding line)
  • lineno (Current row index)

Comments

  • Lines of notes starts with #
